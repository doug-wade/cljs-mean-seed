# {{name}}

A Web application built on Nodejs using Express and AngularJS in Clojurescript.

## Usage

Quick start:
    lein deps
    lein cljsbuild once
    nodejs app.js

A general workflow:
    lein deps
    lein cljsbuild auto
    nodejs app.js
    Repeat:
        Edit some files
        Restart the server
        Look at your browser
